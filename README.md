PDF Print Saver
Goal: Convert normal PDFs into 2-UP cut-and-stack print format to reduce printing cost by 75%.

MVP:
- Upload PDF
- Process using Python backend
- Return 2-UP imposed file with cut line

Stack:
- Frontend: simple HTML/JS or Next.js
- Backend: Python FastAPI + pypdf + reportlab

Deployment:
- Frontend: GitHub Pages (free)
- Backend: Render (free)

Why:
Students & exam aspirants need cheap printing solution.
